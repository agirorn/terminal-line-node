# terminal-line

Prints a line across the terminal

<img src="./assets/cover.png" alt="yardman" style="max-width:600px;"/>

## Installation

```shel
# Using Yarn
yarn add @agirorn/terminal-line

# Using npm
npm install @agirorn/terminal-line
```

## Usage

```shel
yarn -s terminal-line
-----------------------------------------------------------
```
